# Date Solutions Forms

Custom Drupal 11 module for two inline lead-generation forms:

- Contact inquiry form
- Consultation request form

## Block plugins provided

- Date Solutions: Contact inquiry form
- Date Solutions: Consultation request form

## Default email behavior

Both forms send their submission email to the Drupal site email address configured at:

- Configuration -> System -> Basic site settings
